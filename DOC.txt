DISTRIBUTED SYSTEMS PROJECT — MULTI INDEX FOOD SEARCH WITH LLM
==========================================================

CURRENT STATE (END OF PHASE 1)
------------------------------
Project hiện tại là một hệ thống distributed search engine mini cho domain Food.
Người dùng nhập:
- Nguyên liệu
- Tên món
- Keyword

Hệ thống sẽ:
1. Search distributed qua nhiều node
2. Aggregate kết quả
3. Rank kết quả
4. Trả về món ăn phù hợp
5. Detect nguyên liệu còn thiếu

RESPONSIBILITIES
----------------

1. Coordinator
Vai trò:
- Nhận request từ client
- Orchestration
- Gọi các node
- Merge result
- Ranking
- Fault tolerance cơ bản

Coordinator KHÔNG:
- Giữ full dataset
- Tự search data

Coordinator chỉ:
- Route + Aggregate

2. Ingredient Node
Vai trò:
- Giữ inverted index theo ingredient
- Search dishes theo nguyên liệu

Structure:
- ingredient -> list of dishes

Ví dụ:
- egg -> Fried Rice, Omelette
- rice -> Fried Rice

Endpoint: POST /ingredient/search
Input: {"ingredients": ["egg", "rice"]}
Output: [{"dishName": "Fried Rice", "matchedCount": 2}]

3. Dish Node
Vai trò:
- Giữ full dish detail
- Trả thông tin món ăn

Endpoint: GET /dish/{dishName}
Output:
{
  "id": 1,
  "name": "Fried Rice",
  "ingredients": ["rice", "egg", "onion", "soy sauce"],
  "tags": ["asian", "quick"]
}

CURRENT FLOW
------------
STEP 1 — Client Request: POST /search
Body: {"ingredients": ["egg", "rice", "onion"]}

STEP 2 — Coordinator Normalize Query
- Lowercase
- Remove duplicates

STEP 3 — Coordinator -> Ingredient Node
- Coordinator call: http://localhost:8081/ingredient/search
- Ingredient Node: search inverted index, count matched ingredients, return ranked candidate dishes.

STEP 4 — Coordinator -> Dish Node
- Coordinator call: http://localhost:8082/dish/{dishName}
- Dish Node: return full dish detail.

STEP 5 — Coordinator Aggregate
- Calculate missing ingredients
- Calculate score
- Merge final result

CURRENT SCORING
---------------
score = matchedIngredients / totalDishIngredients
Ví dụ: matched = 3, total = 4 -> score = 0.75

CURRENT RESPONSE
----------------
[
  {
    "dish": "Fried Rice",
    "matchScore": 0.75,
    "missingIngredients": ["soy sauce"]
  }
]
